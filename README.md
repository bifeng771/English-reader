# LinguaReader Coach

本地英语阅读学习 App 原型，主界面已简化为沉浸式阅读器。

## 运行

在 PowerShell 中进入本目录后运行：

```powershell
.\run-local.ps1
```

然后打开：

```text
http://127.0.0.1:4179/
```

手机在同一 Wi-Fi 下访问时，不要输入手机自己的 `localhost`。运行本地服务后，终端会显示 `Mobile/LAN URLs`，用其中的 `http://电脑IP:4179/` 在手机浏览器打开。若仍打不开，需要允许 Windows 防火墙放行 Node.js 或 4179 端口。

## 功能

- 导入 TXT、PDF、DOCX、EPUB、Markdown、HTML、CSV、SRT/VTT。
- 鼠标选中单词、词组、句子或段落后，自动弹出工具条。
- 工具条支持拖动、固定、关闭，减少遮挡正文。
- 工具条右下角可拖动调节大小。
- 工具条支持翻译、朗读、加入书签、加粗。
- 光标翻译默认开启，鼠标悬停在单词上会显示释义和音标。
- 双击单词会自动选中并朗读。
- 单词选中后自动显示本地释义和美音/英音音标。
- 选择词组、句子或段落后，会自动生成本地连贯草译、音标序列和句子结构解析。
- 翻译源可选择本地、Google、百度；Google/百度通过官方网页入口打开。
- 跟读评分已放到书架下方，上层显示跟读内容，下层显示识别/手输的跟读语音文本并对比评分。
- 文章交流功能已从界面移除。
- 书签可在左侧隐藏/显示，双击书签会定位到文章中的对应片段。
- 顶部支持美音、英音、澳音口音预设，并优先选择系统中的英文 Natural/Online/Neural 音色。
- 即使未检测到英文自然音色，也会提供系统默认美音、英音、澳音和本地已检测音色作为可选项。
- 顶部提供美音试听和英音试听按钮。
- 顶部支持自然、清晰、温和、有感情四种朗读风格。
- 朗读会按句子和标点切分，加入轻微停顿和语调变化，听感比整段机械朗读更自然。
- 可上传一段参考音频，应用会分析语速、音高、响度和节奏，并用于微调本地朗读风格。

音色质量取决于当前浏览器和系统已安装语音。若未检测到英文音色，应用会使用浏览器默认英语发音；安装 Edge/Chrome 的英文 Natural/Online voice 后效果会更好。纯前端可学习参考音频的朗读习惯参数，但不能真正克隆声纹；声纹克隆需要后端语音模型和授权。
