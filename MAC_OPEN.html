Mac 用户打开 LinguaReader

不要直接点击 http://127.0.0.1:4179/。这个地址只有在 Mac 已经启动本地网页服务后才会打开。

推荐方法：不用拖文件夹

  1. 解压压缩包。
  2. 在 Finder 里找到解压后的文件夹，也就是能看到 index.html、app.js、vendor 的那个文件夹。
  3. 选中这个文件夹。
  4. 按住 Option 键，同时右键文件夹，选择“复制 ... 作为路径名”。也可以直接按 Option + Command + C。
  5. 打开 Mac 的“终端”。
  6. 输入 cd ，注意 cd 后面有一个空格，然后粘贴刚才复制的路径，按回车。
  7. 运行下面这行命令：
python3 -m http.server 4179
  1. 终端窗口不要关闭。
  2. 再用 Safari 或 Chrome 打开：
http://127.0.0.1:4179/

更省事的方法

如果 Finder 右键菜单里有“服务”或“快速操作”，可以这样做：

  1. 右键解压后的文件夹。
  2. 选择“服务”或“快速操作”。
  3. 选择“新建位于文件夹位置的终端窗口”。
  4. 终端打开后，直接运行：
python3 -m http.server 4179

启动脚本方法

在终端进入解压后的文件夹后，运行：

zsh start-mac.command

这个方式不需要先执行 chmod。之后可以尝试双击 start-mac.command 启动；若 Mac 提示安全限制，仍用上面的终端命令启动。

想让别人直接点链接打开

需要把整个文件夹部署到公网 HTTPS，例如 Netlify、Vercel、Cloudflare Pages 或 GitHub Pages。部署后得到的 https://... 地址才可以直接发给别人。
